Code
In Game

Summary

  • A game that presses random colors in order within the time limit

Overview

  • Role : Team Leader(coordinate the duration, encourage team member) / merge programs / Control Touch LCD Image with Frame Buffer

  • Linux Ubuntu
  • C, CPP
